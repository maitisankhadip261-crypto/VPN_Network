# 🌐 Stellar VPN Network (Soroban Smart Contract)

## 📌 Project Description
<img width="1914" height="899" alt="image" src="https://github.com/user-attachments/assets/355949aa-54b8-4101-ac76-1618815d62aa" />


Stellar VPN Network is a decentralized VPN ecosystem built on Soroban (Stellar Smart Contracts). It allows users to access secure and private internet connections while enabling node providers to earn rewards for sharing bandwidth.

This project demonstrates how blockchain can be used to build trustless, transparent, and incentivized networking systems.

---

## 🚀 What It Does

- Allows users to pay for VPN services using Stellar-based tokens
- Enables individuals to register as VPN nodes
- Distributes earnings to node providers
- Maintains transparent balances on-chain

The smart contract acts as a decentralized payment and accounting layer for the VPN network.

---

## ✨ Features

- 🔐 Decentralized VPN payment system
- 👨‍💻 Node registration (anyone can become a provider)
- 💰 Token-based usage payments
- 📊 On-chain balance tracking
- 💸 Secure withdrawal system for node operators
- ⚡ Built using Soroban (Stellar smart contracts)

---

## 🛠️ Tech Stack

- Soroban SDK (Rust)
- Stellar Blockchain
- Smart Contracts for decentralized logic

---

## 📦 Smart Contract Functions

| Function        | Description                          |
|----------------|--------------------------------------|
| register_node  | Register a VPN provider node         |
| pay_for_vpn    | Pay for VPN usage                    |
| withdraw       | Withdraw earnings (for nodes)        |
| get_balance    | Check user/node balance              |
